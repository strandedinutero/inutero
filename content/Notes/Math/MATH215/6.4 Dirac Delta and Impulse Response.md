# The Delta Function

The delta function $\delta(t)$ is essentially a pulse and is defined such that for any continuous function $f(t)$:
$$
\int_{-\infty}^\infty \delta(t)f(t)dt =f(0)
$$
and
$$
\int_{c}^d \delta(t)dt = \begin{cases}
1 \quad \text{if } \quad 0 \in [c, d] \\
0 \quad \text{otherwise}
\end{cases}
$$
Essentially this means that $\delta(t)$ is zero for all $t$ but $\infty$ at $t=0$. Think about an actual one off pulse.

Its Laplace transform is just:
$$
\mathcal{L}\{\delta(t-a)\}=\int_{0}^\infty e^{ -st }\delta(t-a)dt=e^{ -as }
$$
Where $a$ is the time at which the pulse is sent, as when $t=a$ $\delta(0)=1$

In particular, its Laplace transform when $a=0$ is just $e^{ -0s }=1$

## How Is It Used?

We can use the delta function to compute the transforms of improper rational functions, for instance:
$$
\mathcal{L}^{-1} \{\frac{s+1}{s}\} = \mathcal{L}^{-1} \{1 + \frac{1}{s}\}= \delta(t) + 1
$$

## Finding The Impulse Response To a System

For a differential equation of the form $Lx = f(t)$ we can think of $f(t)$ as the input and $L(x)$ as the output, then to find the system's response to an impulse we can just plug in the delta function as out input and solve for $x(t)$. For example, consider a mass spring system with $x(0)=0$ and $x'(0)=0$.  
$$
x'' + \omega_{0}^2x=\delta(t) \iff s^2X(s) + \omega_{0}^2X(s) = 1 \iff x(t) = \frac{\sin(\omega_{0}t)}{\omega_{0}}
$$
