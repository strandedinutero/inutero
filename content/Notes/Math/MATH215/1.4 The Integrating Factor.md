#math #math215 #ODEs 

A first order ODE is linear if it is linear in $y$ and $y'$ such that it can be expressed as:

$$
y'+p(x)y = f(x)
$$
Because it is linear, solutions exist for all continuous $p(x)$ and $f(x)$ and the solution is differentiable.

The trick to solving them is to find some function $r(x)$ such that:
$$
r(x)y' + r(x)p(x)y = \frac{d}{dx}[r(x)y]
$$
In essence, we want some function such that when differentiated we get the same function back but multiplied by $p(x)$ such that the above equation holds as per the *product rule*.

This function is called the integrating factor and can be written as:

$$
e^{ \int p(x)dx }
$$
Where $\int p(x)$ is any anti derivative of $p(x)$

Now, multiplying by $r(x)$ we find:

$$
\begin{align}
e^{ \int p(x)dx }y' + e^{ \int p(x)dx }y & =e^{ \int p(x)dx }f(x)  \\
\frac{d}{dx}\left( e^{ \int p(x)dx }y \right) & = e^{ \int p(x)dx }f(x) \\
e^{ \int p(x)dx }y& = \int e^{ p(x)dx }f(x)dx + C \\
y & = e^{ -\int p(x)dx }\left( \int e^{ \int p(x)dx }f(x)dx+ C \right)
\end{align}
$$
