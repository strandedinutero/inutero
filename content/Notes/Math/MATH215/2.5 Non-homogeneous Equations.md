# Non-homogeneous Equations

That is, equations of the form
$$
y'' + py' +qy = f(x) \quad \text{s.t.} \quad f(x)\neq 0
$$
To solve these we can use the superposition principle discussed in [[2.1 Higher Order Linear ODEs]].

In essence, we can first let $f(x)=0$ and find a solution to the homogeneous version, this is called the *complementary solution* $y_c$ (doesnt matter what order we do this). Then, we can find a *particular solution* $y_p$ such that our general solution $y$ is a linear combination of these two solutions:
$$
y=C_{1}y_{c}+C_{2}y_{p}
$$
==Note==, we must *first* get $y = y_c + y_p$ and then solve for constants.
Now, to solve this the trick is basically just to guess a particular solution smartly.

## An Example

Solve the ODE: $y'' + 5y' + 6y = 2x+1$

Let us first find a particular solution $y_p$, we can see that $f(x) = 2x+1$ so lets guess $y_p = Ax + B$. This yields:

$$
\begin{align}
(Ax+B)'' + 5(Ax+B)'+6(Ax+B) & = 2x+1 \\
6Ax + 5A + 6B & = 2x+1 \\
A=\frac{1}{3} \quad \text{and} \quad B = -\frac{1}{9}
\end{align}
$$
Therefore we have that $y_p = \frac{1}{3}x - \frac{1}{9} = \frac{3x-1}{9}$.

Now, to find $y_c$ we solve the homogeneous equation: $y'' + 5' + 6y = 0$. Using reduction of order method from [[2.2 Solving Constant Coefficient 2nd Order ODEs]] we see that this has roots $r=-2$ and $r=-3$ so our complementary solution is:

$$
y_{c} = C_{1}e^{ -2x }+C_{2}e^{ -3x }
$$
So our general solution is just:
$$
y =  C_{1}e^{ -2x }+C_{2}e^{ -3x } + \frac{3x-1}{9}
$$
==Note==, an important thing to remember is that when we have some $f(x) = x^3 + 1$ for example or really any higher order polynomial, we also use the general form polynomial as our guess, which in this case would be $Ax^3 + Bx^2 + Cx + D$. 

### Edge Cases

Sometimes it could be the case that our guess also solves the complementary solution, for example, if we had the ODE: $y''-9y=e^{ 3x }$, we would be inclined to guess $y_p = Ae^{3x}$, but this would solve the homogenous equation as:

$$
(Ae^{ 3x })'' - 9Ae^{ 3x } = 0
$$
To fix this, we need to multiply our guess by $x$ to get rid of duplication. ==Note: we can multiply by x as many times as necessary==. So let $y_p = Axe^{3x}$:

$$
\begin{align}
(Axe^{ 3x })'' + 9(Axe^{ 3x }) & = e^{ 3x } \\
3Ae^{ 3x }+3Ae^{ 3x }+9Axe^{ 3x }-9Axe^{ 3x } & = e^{ 3x } \\
6Ae^{ 3x } = e^{ 3x } \implies A = \frac{1}{6}
\end{align}
$$

Thus $y_p =\frac{1}{6}xe^{ 3x }$ and our general solution is:
$$
y = C_{1}e^{ 3x }+C_{2}e^{ -3x }+\frac{1}{6}xe^{ 3x }
$$
