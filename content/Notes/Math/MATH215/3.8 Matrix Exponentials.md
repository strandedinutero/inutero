# Matrix Exponentials

We can use $e^{ at }$ as a solution to linear equations, so similarly we can use $e^{ tA }$ as a solution to a system of euqations. 

$e^{ tA }$ is defined by the taylor-series:
$$
e^{tA} = I + tA + \frac{t^2A^2}{2!} + \frac{t^3A^3}{3!} + \cdots  
= \sum_{n=0}^{\infty} \frac{(tA)^n}{n!}
$$
Such that $\frac{d}{dt}e^{tA} = Ae^{ tA }$. and at $t = 0$ this just yields the *identity matrix*. We can write the general solution to a system of linear equations $\vec{x'}=A \vec{x}$ using the matrix exponential such that:
$$
\vec{x}(t) = e^{ tA } \vec{c}
$$
## A Is Diagonal

If A is diagonal, the matrix exponential is extremely easy to calculate as we know if a matrix is diagonal then its eigenvalues are just the diagonal entries. This means that for any matrix A such that:
$$
A = \begin{bmatrix}
\lambda_{1} & 0 \\
0 & \lambda_{2}
\end{bmatrix}
\implies e^{ tA } = \begin{bmatrix}
e^{ \lambda_{1}t } & 0 \\
0 & e^{ \lambda_{2}t }
\end{bmatrix}
$$
Which is our general solution.

## A Is Diagonalizable

A matrix $A$ is diagonalizable if we can can construct a diagonal matrix $A$ with:
$$
A = PDP^{-1}
$$
Where $D$ is the diagonal matrix composed of the eigenvalues of $A$ and $P$ is the matrix with columns composed of the eigenvectors of $A$. 

Recall the inverse of a $2 \times 2$ matrix is given by the formula:
$$
A^{-1} = \frac{1}{\det(A)} \begin{bmatrix}
d & -b \\
-c & a
\end{bmatrix}

$$
Then, we can write our general solution $e^{tA}$ in the form:

$$
\vec{x}(t) = e^{ tA } \vec{c}=e^{ tPDP^{-1} } \vec{c} = Pe^{ Dt }P^{-1} \vec{c}
$$
## Fundamental Matrix Solutions

Recall that the fundamental matrix solution to a system $X(t)$ is a matrix with columns $v_{i}e^{ \lambda_{i}t }$ where $\lambda$ and $v$ are the eigenvalues and corresponding eigenvectors of $A$.

Now, $e^{ tA }$ is also a fundamental matrix, but with the special property that at $t=0$ we just get the identity matrix. This means $e^{ tA }$ can be expressed as:
$$
e^{ tA } = X(t)[X(0)]^{-1}
$$

## Why Is This Useful

This clearly takes the same amount of time to compute as the eigenvalue method so why do we need the matrix exponential. Well, because the matrix exponential can be expressed as a taylor-series it means we can use it to approximate solutions.
