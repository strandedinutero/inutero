# Solving Linear Systems of ODEs

A first order linear system of ODEs is a system that is expressed with the vector equation:
$$
\vec{x'}(t)=P(t)\vec{x}(t)+\vec{f}(t)
$$
Where $P(t)$ is a matrix valued function.

Now, when $\vec{f}(t)=0$ the system is homogeneous and we once again can use the principle of superposition discussed in [[2.1 Higher Order Linear ODEs]]. That is, the solution $\vec{x}(t)$ can be written as a linear combination of many other solutions. Namely, however if we have some particular solution $\vec{x}_{p}$, then our general solution has the form $\vec{x}=\vec{x}_{p}+\vec{x}_{c}$ where $\vec{x}_{c}$ is the solution to the homogeneous equation $\vec{x'}=\vec{P}x$.

## Solving Homogeneous Systems of ODEs

Suppose we have some homogeneous system $\vec{x}=P \vec{x}$, we can adopt the method used for single coefficient ODEs by multiplying by $e^{ \lambda t }$ for some unknown constant $\lambda$. So, we multiply by $ve^{\lambda t}$ which recovers:

$$
\lambda v e^{ \lambda t } = Pv e^{ \lambda t  } \iff \lambda v = Pv
$$

In other words, this will require *eigenvalues* and *eigenvectors*.

## Eigenvalues and Eigenvectors Refresher

The eigenvalues of a matrix $A$ are given by the equation $\det(A-\lambda I)=f(\lambda)=0$. Where the eigenvectors of a matrix $A$ are given by solving for $v$ for a particular eigenvalue in the equation $(A-\lambda I)\vec{v}=0$. 

==Note,== for upper/lower diagonal matrices its eigenvalues are simply just the diagonal entries of the matrix. 

==Recall,== the determinant of a $2 \times 2$ matrix is given by $det(A)=ab-bc$. Where for a $3 \times 3$ matrix we can use cofactor expansion.

## The Eigenvalue Method

Given some system $\vec{x'}= P \vec{x}$, we can find the eigenvalues and eigenvectors of the matrix $P$ in order to construct the general solution:
$$
\vec{x}(t)=c_{1}v_{1}e^{ \lambda_{1}t }+c_{2}v_{2}e^{\lambda_{2}t}  + \cdots
$$

and the fundamental matrix solution is given by:

$$
X(t) = \begin{matrix}
[v_{1}e^{ \lambda_{1}t } \cdots v_{n}e^{ \lambda_{n} }t]
\end{matrix}
$$

## Complex Eigenvalues

We can use Euler's formula from [[2.2 Solving Constant Coefficient 2nd Order ODEs]] in order to express the general solution to a system with complex eigenvalues. First, note that if we know $\lambda=a+ib$ is an eigenvalue, then so is $\lambda=a-ib$, similarly their eigenvectors will be identical except for the fact that one will have positive $i$ and the other negative $i$.

## Repeated Eigenvalues

If a system has repeated eigenvalues, in order to find a solution we really only care about having $n$ linearly independent eigenvectors for an $n \times n$ matrix. So the general solution wont change and we can use $\lambda$ more than once.

## Defective Eigenvalues

If an $n \times n$ system has less than $n$ linearly independent eigenvectors, then to express the general solution we need to find another vector $v_2$ such that:

$$
(A-\lambda I)v_{2}=v_{1}
$$
Then, we can express the solution as follows:

$$
x(t)=v_{1}e^{ \lambda t }+(v_{2}+v_{1}t)e^{ \lambda t }
$$
