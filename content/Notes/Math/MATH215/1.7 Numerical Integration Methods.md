# Euler's Method

For some differential equation $f(x, y$) we can approximate a solution using some initial condition $y(x_0) = y_0$. Choose some step size $h$ and your initial condition.

1. Take $x_0$ and $y_0$ and compute the slope $f(x_0, y_0)$ 
2. The next $y$ value $y_1 = y_0 + h \cdot f(x_0, y_0)$ 
3. The next $x$ value $x_1 = x_0 + h$

Generally:
$$
\begin{align}
y_{n+1} & = y_{n} + hf(x_{n}, y_{n}) \\
x_{n+1} & = x_{n} + h
\end{align}
$$
The line segments obtained from this approximate the original function graph.

### Error

The error in this approximation is given by:
$$
\text{ error} = |\text{actual(y)} - \text{approximate(y)} |
$$
## Improved Euler's method

Start with initial points $(x_0, y_0)$ then compute the slope $k_1 = f(x_0, y_0)$, now instead of our new point being at $(x_0 + h, y_0 + k_1h)$ compute the slope at this point first and call it $k_2$, now take the average of $k1$ and $k_2$ hoping this slope is a better approximation as it is now a second order approximation. 

Generally:

$$
\begin{align}
y_{n+1} & = y_{n} + \frac{k_{1}+k_{2}}{2}h  \\
x_{n+1} & = x_{n} + h
\end{align}
$$
